The start of a great adventure!


            Console.Log("Hello World!");

Welcome everyone to the start of a great new adventure! It just occurred to me to start recording my adventures through building my website as well as a bunch of other projects that I will have and inevitably abandon. I have just finished setting up my devlog system. It includes it's own custom (janky and poorly written) template. Now with features like highlight.js and Disqus. The general plan is to write a devlog every week AND once I make some decent progress. I'm not too sure about what format to go with exactly, but I'll run with this format and see where it leads. I like the code highlighting plugin I have so far, but I'm not too sure on Disqus. The lack of actual usage aside, the dashboard is a bit confusing. I have to compliment both plugins on how easy it was to import and setup though. I also wanted to thank Kenny Park from Purdue IGDC club for letting me steal all his ideas.

A Portfolio

As I mentioned earlier, I wanted this website to be a professional portfolio of sorts where I can show off my resume and the projects I have worked on to curious job recruiters that accidentally clicked on the link when trying to recycle my resume. As of the time of writing, I am currently looking for a paid Summer 2022 internship anywhere in the US. (hint hint...)
