Formatting pitfalls

As usual, doing anything other then directly copy-pasting a previous page involves some difficulties and pitfalls. For one, making a code block too horizontally wide will mess up the alignment of the entire page. Another difficulty is messing up Disqus's PAGE_URL variable to use the full relative path will just prevent it from appearing.
