Projects

CSV to OFX

Github repo

CSV to OFX is a tool that that converts financial documents in CSV (comma seperated variable) format to OFX (Open Financial Exchange) Format. This is used to aid in the importing of online banking statements to an old accounting program called Microsoft Money 2010.

Unity GameEngine was used to create the program. GitHub was used for Version Control.

AquaCat

Itch.io Link

AquaCat is a side-scrolling arcade minigame, based off of a cover of Nyancat. It was built in Unity GameEngine and has been ported to Windows, Android, and WebGL (itch.io).

Pekora Quest

PekoraQuest is a grand 2D RPG that was a fangame for Hololive productions. It has a playtime of over 2 hours and many features seen in classic games of the genre like Earthbound. Pekora Quest was made in Gamemaker 2, a derivative of C.

(WIP)Virtual Incremental

Virtual Incremental is an Incremental genre game heavily inspired by the game Progress Knight. It makes heavy usage of progress bars to represent progress in job and skill training. Written in Unity GameEngine.

(WIP)SmartFish

SmartFish is an attempt at creating an evolutional machine-learning program where the "tank" of fish will slowly learn how to best find their food. Written in Unity GameEngine.
